# Warehouse Route Optimization (Python)

## Overview
This project simulates a warehouse picking system and calculates the most efficient route using Dijkstra’s Algorithm.

## Problem
Warehouse picking routes are often inefficient, leading to wasted time and reduced productivity.

## Solution
A graph-based warehouse layout was created, and Dijkstra’s Algorithm was used to determine the shortest path between locations.

## Tools Used
- Python
- Data Structures (Graphs)
- Algorithms (Dijkstra’s Algorithm)

## Key Features
- Graph-based warehouse model
- Shortest path calculation
- Optimized route output
